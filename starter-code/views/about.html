Acerca de esta página

Armé esta página de manera rápida como parte del bootcamp de Web Development (Part Time Oct 2018) lanzado por Ironhack en la Ciudad de México.

El objetivo es demostrar lo fácil que es lanzar una página web completa, con archivos y todo, servida a través de Node y Express.

En realidad fue tan fácil como parecía.

Me puedes encontrar en Github si deseas.
